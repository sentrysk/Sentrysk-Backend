# Dashboard

# Agents
In this section you can create,update,delete,list **Agents**

Endpoint prefix: **/agent**

| Action      | Endpoint | Method |
| ----------- | ----------- | ----------- |
| Get         | /           | GET |
| Get by ID   | /id         | GET |
| Create      | /register   | POST |
| Delete      | /<id>       | DELETE |
| Update      | /<id>       | PUT |
